Política de Privacidade

eProc Calendar — Extensão para Google Chrome

Última atualização: 14 de março de 2026

1. Introdução

A extensão eProc Calendar permite que profissionais do Direito adicionem audiências e prazos processuais do sistema eProc diretamente ao Google Calendar com um clique. Esta Política de Privacidade descreve como a Extensão trata os dados dos usuários.

2. Dados coletados e utilizados

A Extensão acessa os seguintes dados exclusivamente para cumprir sua funcionalidade:

  • Dados de páginas do eProc: A Extensão lê informações exibidas nas páginas do sistema eProc (números de processos, nomes de partes, datas de audiências e prazos) para montar os eventos do calendário. Esses dados são processados localmente no navegador do usuário e não são enviados a nenhum servidor externo.
  • Token de autenticação OAuth do Google: Ao autorizar a Extensão, um token de acesso é obtido via fluxo OAuth 2.0 do Google. Esse token é armazenado localmente no navegador do usuário (via chrome.storage.local) e é utilizado exclusivamente para criar eventos no Google Calendar do usuário.
  • Configurações do usuário: Preferências como antecedência em dias, cor dos eventos e emails de convidados são armazenadas localmente no navegador (via chrome.storage.sync).

3. Dados NÃO coletados

A Extensão NÃO:

  • Coleta, armazena ou transmite dados pessoais para servidores externos
  • Utiliza cookies de rastreamento ou ferramentas de analytics
  • Compartilha dados com terceiros
  • Armazena dados processuais fora do navegador do usuário
  • Acessa ou modifica eventos existentes no Google Calendar — apenas cria novos eventos

4. Uso da API do Google

A Extensão utiliza a API do Google Calendar com o escopo calendar.events, que permite exclusivamente criar eventos no calendário do usuário. A Extensão:

  • Não lê, modifica ou exclui eventos existentes do calendário
  • Não acessa outros serviços do Google além do Google Calendar
  • Segue a Política de Dados do Usuário dos Serviços de API do Google, incluindo os requisitos de Uso Limitado

5. Armazenamento de dados

Todos os dados são armazenados localmente no navegador do usuário:

  • Token OAuth: armazenado em chrome.storage.local, acessível apenas pela Extensão
  • Configurações: armazenadas em chrome.storage.sync, sincronizadas entre dispositivos do usuário via conta do Chrome

Nenhum dado é armazenado em servidores externos.

6. Permissões da extensão

A Extensão solicita as seguintes permissões do Chrome:

  • identity: para autenticar o usuário com sua conta Google via OAuth 2.0
  • storage: para salvar configurações e o token de acesso localmente
  • scripting: para injetar botões de ação nas páginas do eProc
  • Acesso a sites do eProc: para ler informações de audiências e prazos exibidas na tela
  • Acesso a googleapis.com: para criar eventos no Google Calendar

7. Revogação de acesso

O usuário pode revogar o acesso da Extensão a qualquer momento:

  • Removendo a Extensão do Chrome (Menu > Extensões > Gerenciar extensões)
  • Revogando o acesso em Configurações de Segurança da Conta Google

8. Segurança

A Extensão adota as seguintes medidas de segurança:

  • Todas as comunicações com o Google são feitas via HTTPS
  • O token OAuth é armazenado de forma segura no armazenamento interno do Chrome, inacessível a outros sites ou extensões
  • Nenhum dado sensível é transmitido para servidores de terceiros

9. Alterações nesta política

Eventuais alterações nesta Política de Privacidade serão publicadas nesta página com a data de atualização revisada. Recomendamos que o usuário revise esta política periodicamente.
