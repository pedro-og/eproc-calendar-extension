eProc Calendar

Extensão gratuita para Google Chrome que adiciona audiências e prazos processuais do sistema eProc diretamente ao seu Google Calendar com um clique.

O que é o eProc Calendar?

O eProc Calendar é uma extensão para Google Chrome desenvolvida para advogados, promotores e demais profissionais do Direito que utilizam o sistema eProc (e-Proc TRF4, JFRS, e plataformas correlatas).

Com um único clique, a extensão lê as audiências e prazos processuais exibidos na tela do eProc e cria eventos no seu Google Calendar, eliminando a necessidade de inserção manual de dados.

Funcionalidades

📅

Audiências para o Calendar

Exporte audiências futuras listadas no eProc para o Google Calendar com um clique.

⏰

Prazos processuais

Crie eventos de prazos abertos diretamente a partir das páginas de processos.

🔔

Lembretes automáticos

Configure a antecedência dos lembretes e receba notificações antes do prazo.

🎨

Personalização

Escolha a cor dos eventos e adicione convidados automaticamente.

Como funciona

  1. Instale a extensão pelo Chrome Web Store.
  2. Autorize o acesso à sua conta Google (somente criação de eventos no Calendar).
  3. Acesse as páginas de audiências ou prazos no sistema eProc.
  4. Clique no botão "Adicionar ao Calendar" exibido pela extensão.
  5. O evento é criado automaticamente no seu Google Calendar.

Quais dados a extensão acessa e por quê?

A extensão acessa apenas os dados estritamente necessários para sua funcionalidade. Veja a lista completa:

Dado Finalidade Armazenamento
Conteúdo das páginas do eProc (números de processos, nomes de partes, datas) Montar os eventos do calendário com as informações corretas Processado localmente no navegador — nunca enviado a servidores externos
Token OAuth do Google Autenticar o usuário para criar eventos no Google Calendar Armazenado localmente no Chrome (chrome.storage.local) — nunca transmitido
Preferências do usuário (cor dos eventos, antecedência, convidados) Personalizar os eventos criados conforme configuração do usuário Sincronizado pelo Chrome (chrome.storage.sync) entre dispositivos do usuário

Nenhum dado pessoal ou processual é enviado a servidores externos. Todo o processamento ocorre localmente no seu navegador. A extensão não coleta analytics, não usa cookies de rastreamento e não compartilha informações com terceiros.

Permissões solicitadas ao Chrome

A extensão solicita as seguintes permissões, cada uma com finalidade específica:

  • identity Autenticar o usuário com sua conta Google via OAuth 2.0, necessário para criar eventos no Google Calendar.
  • storage Salvar o token de acesso e as preferências do usuário localmente no navegador.
  • scripting Injetar os botões de ação ("Adicionar ao Calendar") nas páginas do eProc.
  • Acesso ao eProc Ler as informações de audiências e prazos exibidas nas páginas do sistema eProc para montar os eventos.
  • googleapis.com Comunicar-se com a API do Google Calendar (escopo calendar.events) para criar eventos.

Uso da API do Google

A extensão utiliza a Google Calendar API com o escopo restrito https://www.googleapis.com/auth/calendar.events, que permite exclusivamente:

  • Criar novos eventos no calendário do usuário

A extensão não lê, modifica, exclui ou lista eventos existentes. Não acessa nenhum outro serviço do Google.

O uso da API está em conformidade com a Política de Dados do Usuário dos Serviços de API do Google, incluindo os requisitos de Uso Limitado.
